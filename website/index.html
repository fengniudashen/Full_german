Core Features

全方位德语学习功能矩阵

覆盖听、说、读、写、词汇、语法六大维度，构建完整学习闭环

🎧

智能听写系统

基于 Wagner-Fischer 编辑距离算法的逐词智能对比，绿色正确、黄色小错、红色错误。支持变速播放 0.5x-2.0x、区间循环、专注模式。

逐词对比变速播放区间循环专注模式
🗣️

口语练习 + AI 教练

录音后自动语音转写（本地 Whisper / 云端 API），与原文逐词对比计算准确率。AI 发音教练提供 IPA 注音、重音语调、连读指导。

录音转写准确率评估IPA 注音发音教练
📖

沉浸式阅读

全文连续展示，点击任意单词即刻查词。三级词典：本地简易词库 → FreeDict 363K 词条 → 在线 API → AI 深度解析。

点击查词三级词典字体缩放AI 解析
🎤

卡拉OK式跟读

实时高亮当前播放句子，自动滚动追踪。默认 0.7x 慢速适合跟读，点击任意句子跳转播放。泛听模式支持纯听不看文字。

实时高亮自动滚动9级变速泛听模式

多模态练习

完形填空自动挖空、句子排序拖拽重组、AI 听力理解 5 题选择、写作批改 CEFR 评级。8 种练习模式全面覆盖。

完形填空句子排序听力理解写作批改
🧠

艾宾浩斯复习

错词自动入库，艾宾浩斯间隔 (1h→8h→1d→2d→4d→7d→15d→30d) 科学复习。闪卡翻转动画，句海拾遗随机抽练。

Methodology

「尚雯婕法」学习四步曲

经实践验证的高效德语学习方法论，每一步都有 AI 深度辅助

1

听写

逐句听写，智能对比。AI 纠错入库，精准定位薄弱点。

2

精析

AI 语法分析、查词释义、翻译对照。深入理解每个细节。

3

跟读

卡拉OK式跟读，变速训练。AI 发音教练实时指导。

4

背诵

间隔复习，闪卡强化。完形填空、句子排序巩固记忆。

Import

多元内容导入系统

YouTube、播客 RSS、本地文件、字幕解析 — 你的德语素材，轻松导入

▶️

YouTube 视频导入

自动下载 yt-dlp，支持任意 YouTube 视频/播放列表。智能解析 VTT 字幕（含逐词时间戳），一键创建学习项目。

🎙️

播客 RSS 导入

预设 Slow German、Deutsche Welle 等优质播客。支持任意 RSS 源，下载音频后本地/云端 Whisper 自动转写。AI 智能分句优化。

📁

本地文件 & 字幕

导入 MP3/M4A/WAV 音频 + TXT/SRT/VTT 文本。VTT 智能解析 YouTube 自动字幕逐词时间戳，自动按标点拆句。
