# DeutschFlow — 全功能文档

> 一款融合 AI 深度辅助与「尚雯婕法」（听写→精析→跟读→背诵）学习方法论的全平台德语学习应用。

**技术栈**: Flutter + Dart · Provider · Drift (SQLite) · Material 3 Design  
**平台**: Windows (主) · Android · iOS · macOS  
**AI 支持**: 8 大 LLM 提供商 · 本地 Whisper 语音识别  
**词典**: 内置 FreeDict 德英词典 363,916 词条

## 二、核心听写系统

### 2.1 项目管理

- 自定义项目名称
- 音频导入: MP3/M4A/WAV 等格式 (FilePicker)
- 德语原文导入:
  - 直接粘贴文本
  - 从文件导入: **TXT、SRT 字幕、VTT 字幕**
  - VTT 智能解析: 支持 YouTube 自动字幕 `<c>` 逐词时间戳，自动按标点拆句
- 项目卡片: 名称、状态标签（待标注/可听写/进行中/已完成）、标注进度条、听写进度条、准确率环形图
- 网格自适应: 1/2/3 列自动切换

### 2.2 时间轴标注

- 音频波形可视化 (`flutter_audio_waveforms`)
- 播放控制: 播放/暂停、快进/退 2s/5s、进度拖拽
- 为每个句子精确标记音频起止时间点
- 批量操作、完成标注状态切换

### 2.3 听写练习（核心）

#### 基础听写
- 逐句播放 → 用户输入 → 智能核对
- 快捷特殊字符输入: `ä ö ü ß Ä Ö Ü ẞ`
- 键盘快捷键: F5 播放、Ctrl+Enter 核对
- 自动跳转下一句（可配置）

#### 音频控制
- 变速播放: 0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x
- 整句循环播放
- **区间循环**: RangeSlider 设置自定义循环区间（精确到句内片段）

#### 智能对比系统
基于 **Wagner-Fischer 编辑距离算法** 的逐词对比:
- ✅ 绿色: 完全正确
- 🟡 黄色: 小错（大小写、标点、ß→ss）
- ❌ 红色: 拼写错误、漏词、多词
- 大小写折叠、标点剥离、ß→ss 等容错处理

#### 对比结果弹窗
- 详细逐词对比可视化
- 正确率统计
- 错误词自动录入错词本
- 精彩句子一键收藏

#### 专注模式
- 隐藏进度和提示信息，沉浸式听写

#### 快速切换练习模式
- 🎧 泛听 → 🗣️ 跟读 → 🎤 口语 → 📖 阅读
- 练习菜单: 完形填空、句子排序、听力理解、写作批改

## 三、多模态练习矩阵

### 3.1 泛听模式

> 尚雯婕法 Step 1：「不看原文，纯靠耳朵」

- 纯听模式: 默认隐藏文字
- 当前句高亮 + 自动追踪播放位置
- 全文循环、9 级变速
- 听播次数统计

### 3.2 跟读模式

> 尚雯婕法 Step 3：「全文背诵与原声同速共振」

- **卡拉OK式高亮**: 实时跟踪当前句
- 自动滚动到当前播放位置
- 默认 0.7x 慢速适合跟读
- 点击任意句子跳转播放

### 3.3 口语练习 🎤🤖

- 录音: `record` 包录制用户口语
- 语音转写:
  - **在线**: OpenAI Whisper API
  - **本地**: whisper.cpp (离线，无需网络)
- 转写文本与原文逐词对比，计算准确率
- 准确率环形图可视化
- **AI 发音教练**:
  - 逐词 IPA 注音
  - 重音与语调分析
  - 连读与吞音指导
  - 中国学习者发音难点
  - 慢速跟读分段

### 3.4 阅读模式 📖

- 全文连续展示
- **点击任意单词即刻查词**
- 字体缩放: 12-32px
- 三级词典查询:
  1. 本地简易词库 (60+ 常用词，中文释义)
  2. 内置 FreeDict 德英词典 (363,916 词条)
  3. 在线 dictionaryapi.dev API
- AI 深度词汇解析

### 3.5 完形填空

- 自动从句子中随机挖空 1-3 个词
- 智能选词: 优先≥2字符实词，避免标点
- 即时正确/错误反馈（绿/红）
- 正确率实时统计 + 进度条

### 3.6 句子排序

- 打乱句子中的单词顺序
- 点击选词放入答案区 → 点击退回
- 全部放完自动判定
- 逐词对比（绿/红高亮）
- 确保打乱后与原句不同

### 3.7 听力理解 🤖

- AI 根据项目文本自动生成 **5 道选择题**
- 中文出题，即时批改
- 每题附 AI 解释
- 完成后显示总成绩

### 3.8 写作批改 🤖

- 自由德语写作 + 8 个预设话题
- AI 全面批改:
  - CEFR 等级评估 + 百分制评分
  - 逐条错误修正 (❌原文 → ✅修正)
  - 修改后完整文本
  - 语法点评、词汇建议、写作建议
- 实时字数统计

## 四、AI 深度功能（12+ 项）

### 4.1 AI 德语助手（6 种分析模式）

| 模式 | 功能 |
|------|------|
| 语法分析 | 句型结构、动词分析、成分标注、语法规则 |
| 查词释义 | 本地词典 + AI 深度解析（词性/变形/搭配/例句） |
| 翻译 | 直译 + 意译 + 关键词汇 + 文化注释 |
| 片段解析 | 含义、语法分析、固定搭配、类似表达 |
| 发音指导 | IPA 注音、重音语调、连读吞音、中国人难点 |
| 自由提问 | 任意德语学习问题 |

### 4.2 德语对话 🗣️

- **8 种角色扮演场景**: 自由对话、咖啡馆点餐、超市购物、看医生、火车站问路、租房看房、自我介绍、工作面试
- AI 扮演角色（服务员/医生/房东等）
- 实时纠错:【纠正】标签指出语法错误
- 引导提示:【提示】标签建议下一句

### 4.3 每日德语 📅

- **5 大板块**: 今日单词 · 今日句子 · 语法小贴士 · 迷你练习(3题) · 文化小知识
- CEFR 级别切换: A1-C1 五档
- 基于日期生成，可重新生成

### 4.4 文本难度分析 📊

- **本地实时统计**: 词数、句数、独立词数、平均词长、平均句长、词汇多样性百分比
- **AI CEFR 评估**: A1-C2 级别判定 + 理由
- 高级词汇标注、语法复杂度分析、学习建议

### 4.5 双语对照翻译

- 双向: 德→中 / 中→德
- AI 句对齐平行翻译
- 编号展示，支持长文本

### 4.6 AI 造句练习

3 种模式:
- **单词造句**: 词汇卡片 + 5 例句 + 搭配用法 + 同反义词 + 记忆技巧 + 迷你测试
- **语法练习**: 规则说明 + 例句 + 练习题 + 常见错误
- **主题作文**: 核心词汇表 + 范文 + 句型模板 + 讨论问题
- 快捷建议按钮: 预设词汇/语法点/主题

### 4.7 AI 学习规划 🧠

- **数据驱动**: 自动收集用户学习数据（练句数、连续天数、项目数、错词分布）
- 生成 5 大板块:
  - 📊 学习状况诊断
  - 🎯 今日学习计划 (15-30分钟)
  - 📝 重点复习清单 (5词 + 记忆技巧)
  - 💡 提升建议 (3条)
  - 🏆 本周目标

### 4.8 支持的 AI 提供商（8 个）

| 提供商 | 默认模型 |
|--------|----------|
| DeepSeek | deepseek-chat |
| ChatGPT (OpenAI) | gpt-4o |
| Claude (Anthropic) | claude-sonnet-4-20250514 |
| MiniMax | MiniMax-M2.7 |
| GLM (智谱) | glm-4-flash |
| Qwen (通义千问) | qwen-plus |
| Doubao (豆包) | doubao-1.5-pro-32k |
| 自定义 | OpenAI 兼容 API，可配置 URL/模型/Key |

每个提供商独立配置 API Key、模型名、自定义 URL。

## 五、内容导入系统

### 5.1 YouTube 视频导入

- 自动下载/更新 yt-dlp
- 预设播放列表: TAGESSCHAU 100 Sekunden、TAGESSCHAU 完整版
- 支持任意 YouTube 视频/播放列表 URL
- 优先下载纯音频 (m4a)，回退含视频格式
- 自动下载德语字幕 (自动生成/人工，VTT 格式)
- 下载完成后自动解析字幕 → 创建学习项目

### 5.2 播客 RSS 导入

- 预设播客:
  - Slow German (慢速德语)
  - Deutsche Welle – Langsam gesprochene Nachrichten
  - Coffee Break German
- 支持任意播客 RSS 源
- RSS 解析: title, description, enclosure
- 下载 MP3 → 输入原文 → 创建项目

### 5.3 本地文件导入

- 音频: MP3, M4A, WAV 等
- 文本: TXT, SRT, VTT

### 5.4 字幕解析能力

- **VTT**: 支持 YouTube 自动字幕 `<c>` 逐词时间戳，自动按标点拆句
- **SRT**: 标准 SRT 格式解析
- 过滤声音标注 `[jubel]`, `[Musik]`
- 跳过冻结帧字幕 (<50ms)
- 处理序号 "21." 等特殊情况

## 六、复习与记忆系统

### 6.1 错词本

- 听写时红色错误自动入库
- 按项目/掌握状态筛选
- 错词卡片: 错误形式(红) → 正确形式(绿) + 原文句子 + 来源项目 + 日期
- 标记已掌握/取消掌握
- 导出:
  - **CSV**: 错误形式、正确形式、原文、来源、日期、掌握状态
  - **Anki TSV**: HTML 格式卡片（正面=单词, 背面=上下文），自动标签

### 6.2 闪卡复习（艾宾浩斯间隔）

- 间隔: 1h → 8h → 1d → 2d → 4d → 7d → 15d → 30d
- 翻转动画: 正面错误拼写 → 背面正确形式 + 上下文
- "已掌握" (调度下次复习) / "还不熟" (重置短间隔)
- 卡片随机打乱
- 全部掌握时 🎉 庆祝

### 6.3 句海拾遗 / 词海拾遗

- **句海拾遗**: 随机抽取 10 条句子，显示来源项目，可复制/跳转 AI 分析
- **词海拾遗**: 随机抽取 10 个未掌握错词，显示错误/正确形式和上下文
- 🎲 "换一批" 刷新

### 6.4 金句本

- 从听写/跟读中收藏的精彩句子
- 点击播放对应音频片段
- 复制/取消收藏

## 一、 功能扩展建议（Features to Add）

### 1. 专属「盲听背诵/断句挑战」模式

针对高阶学习者“盲听背诵”的需求，现有的听写是基于键盘输入的，可以扩展一个**纯语音交互的背诵检测模块**：

* **流程**：不看文本，听完一段原声（如新闻）后，直接点击录音进行整段背诵。
* **AI/本地转写比对**：利用本地 Whisper 将背诵音频转为文本，再通过 `TextComparator` 与原文比对。
* **语气词与停顿检测**：在对比结果中，不仅标注错词，还要专门标出用户在何处出现了长时间停顿、卡顿或诸如 *äh*, *öhm* 的语气词，帮助优化语流和表达流利度。

### 2. 德语高级语言特征（C1/C2 级）AI 自动提取

新闻和高阶语料中充斥着德语特有的复杂句式，仅靠普通的查词是不够的。可以在 AI 助手或阅读模式中加入：

* **功能动词结构（Nomen-Verb-Verbindungen）自动识别**：例如自动高亮 `in Aussicht stellen`（展现出…的前景）或 `unter Beweis stellen`（证明），并给出替换为普通动词的对比。
* **小品词（Modalpartikeln）语境解析**：针对 *ja*, *denn*, *doch*, *wohl* 这些改变句子情绪但极难掌握的小品词，让 AI 一键解析当前句中它的“弦外之音”。
* **框形结构（Satzklammer）视觉连线**：在阅读或跟读模式下，提供一个开关，能用虚线将变位动词与第二部分（可分前缀、不定式或分词）连接起来，一眼看清德语长难句的骨架。

### 3. 音高与语调视觉化比对（Pitch Contour）

口语教练目前基于文本转写比对，如果能加入“视觉化语流比对”会更惊艳：

* **做法**：提取原声音频的基频数据（F0 曲线）并绘制出来，用户跟读后，将用户的音高走势和节奏曲线重叠在原声曲线下方。
* **效果**：不用看复杂的语音学报告，用户一眼就能看出自己的德语升降调、重音放得对不对，这对于模仿专业新闻主播的语调极有帮助。

### 2. 更鲁棒的德语文本模糊对比（Fuzzy Matching）

在听写系统的编辑距离算法中，德语有其特殊性。目前的容错处理（ß $\rightarrow$ ss）很好，但建议引入更弹性的扣分机制：

* **语法性轻微错误不致命**：例如弱变化名词词尾漏了 `-n`（如 `den Menschen` 写成 `den Mensch`），或者形容词词尾强弱变化写错（`-em` 写成 `-en`）。在最终评分时，可以将这类“语法格错误”与“完全不沾边的错词/漏词”区分开，黄色标签可以再细分一下，避免严重挫伤积极性。

### 3. 自动化管线（Automation Pipeline）

针对 YouTube 和播客 RSS 导入，可以增加一个「订阅自动下载」的后台任务（在桌面端尤为适用）：

* 用户一打开软件，今天的“新鲜语料项目”就已经躺在列表里，点击即练，免去手动粘贴 URL 的等待。

### 一、 针对“德语痛点”的深度定制（杀手锏功能）

德语的词性、格、长复合词和框形语序是学习者的最大噩梦，现有的通用 AI 往往无法精准解决这些痛点。

1. **🧩 复合词自动拆解器 (Compound Word Splitter)**
   - **痛点**：遇到 `Donaudampfschifffahrtsgesellschaft` 这种词，初学者直接崩溃。
   - **功能**：在阅读或查词时，点击长复合词，AI 自动将其拆解为词根（如 `Donau` + `Dampf` + `Schiff` + `fahrts` + `gesellschaft`），并标注每个词根的词性和含义。
2. **🎨 词性与格的“色彩编码” (Color-Coding System)**
   - **痛点**：`der/die/das` 和四格（N/D/A/G）难以形成肌肉记忆。
   - **功能**：提供一个“语法透视”开关。开启后，文本中的名词按词性高亮（如：阳性=蓝色，阴性=红色，中性=绿色），介词后的名词按格高亮。通过视觉刺激培养语感。
3. **🌉 框形结构与语序可视化 (Syntax Arc Visualizer)**
   - **痛点**：德语的“动词第二位”和“从句动词末尾”让听力理解困难。
   - **功能**：利用 AI 生成句法弧线图，在精析页面用弧线连接助动词与实义动词（框形结构），或标出主从句的动词位置，帮助拆解长难句。
4. **🔗 固定搭配与介词自动提取 (Nomen-Verb & Präpositionen)**
   - **痛点**：德语有大量固定搭配（如 `eine Entscheidung treffen` 而不是 `machen`）和绑定介词（`warten auf + Akk`）。
   - **功能**：AI 自动扫描导入的课文，提取出所有的“动词+介词”和“固定搭配”，自动生成**完形填空专项训练**，这比随机挖空价值高百倍。

### 二、 核心功能的体验升级 (UX/UI 打磨)

1. **🧠 听写系统的“语义级”容错 (Semantic Matching)**
   - **现状**：Wagner-Fischer 算法基于编辑距离，只能做字面比对。
   - **改进**：如果原文是 *"Ich habe Hunger"*，用户听写成 *"Ich bin hungrig"*，系统应调用轻量级 AI 或同义词库，判定为 **“语义正确（但表达不同）”**，给予黄色提示而非红色错误，避免挫败感。
2. **⌨️ 移动端专属“德语悬浮键盘”**
   - **现状**：手机端频繁切换系统键盘输入 `ä, ö, ü, ß` 非常打断心流。
   - **改进**：在输入框上方增加一个极简的**德语特殊字符快捷条**，或者支持**长按元音字母**直接弹出变元音选项。
3. **📖 词典的本地化与深度化**
   - **现状**：FreeDict 是德英词典，对中国学生不够直观。
   - **改进**：引入 **德汉词典**（如开源的 DeHan 字典），或爬取 **Wiktionary 德语版** 的数据。德语词典必须包含：**复数形式**、**过去式/第二分词**、**支配的格**，这比单纯的释义重要得多。
4. **🎙️ 影子跟读 (Shadowing) 的音高/节奏对比**
   - **现状**：口语练习偏向 ASR（文本转写对比）。
   - **改进**：引入 **Pitch Contour（音高曲线）** 对比。将原声的语调波形和用户录音的波形重叠显示，让用户“看”到自己的起伏是否和母语者一致，专治“中式德语”平淡语调。

### 三、 AI 潜力的进一步挖掘

1. **🪄 文本“降维/升维”改写 (Text Simplification)**
   - **场景**：导入了一篇 C1 级别的《明镜周刊》新闻，但用户只有 B1 水平。
   - **功能**：一键让 AI 将文本**降维改写**为 B1 版本（替换生僻词、拆分长难句、简化语法），同时保留原音频。反之，也可将简单文本升维，学习高级表达。
2. **🗺️ 情景词汇联想图谱 (Mind Map)**
   - **功能**：在“词海拾遗”或查词时，AI 自动生成该词的**思维导图**（如学 `Umwelt`，延伸出 `Umweltschutz`, `Verschmutzung`, `Klimawandel`），一键打包加入闪卡复习。
3. **🎭 基于当前课文的“动态 Role-play”**
   - **现状**：预设了 8 种对话场景。
   - **改进**：利用当前正在学的项目文本作为 Context。比如课文是关于“去邮局寄包裹”，AI 自动扮演邮局职员，用**课文里的生词和句型**与用户进行开放式对话，实现“学以致用”。
4. **📝 写作批改的“同义替换”建议**
   - **改进**：除了纠错，AI 应提供“高级词汇替换”建议（如把 `machen` 替换为 `durchführen`，把 `gut` 替换为 `hervorragend`），并附带 CEFR 等级提示。

### 四、 学习闭环与生态扩展

1. **📸 OCR 实景德语捕捉 (Snap & Learn)**
   - **功能**：调用手机摄像头，拍摄德语原版书、菜单、路牌、商品包装。OCR 提取文本后，直接查词或加入“生词本”，打通线下与线上的学习闭环。
2. **☁️ 多端数据同步 (Cloud Sync)**
   - **现状**：Drift (SQLite) 是本地数据库。
   - **改进**：如果用户同时使用 Windows 和手机，需要引入 **Supabase / Firebase / 坚果云 WebDAV** 同步机制，确保听写进度、错词本、金句本跨端无缝衔接。
3. **🇨🇳 国内生态适配 (针对中国大陆用户)**
   - **痛点**：YouTube 导入和某些播客 RSS 存在网络门槛。
   - **改进**：增加 **Bilibili 视频解析**（很多优质德语学习资源在B站），以及**小宇宙/喜马拉雅**播客源导入。
4. **🔊 高质量 TTS 引擎集成 (针对纯文本导入)**
   - **场景**：用户粘贴了一段 TXT 文本，没有音频。
   - **改进**：集成 **Edge TTS** 或 **Azure TTS**，提供多种德语口音（德国、奥地利、瑞士）和性别选择，自动生成音频并应用“尚雯婕法”流程。

### 五、 技术与工程架构建议

1. **离线 AI 模型探索 (On-Device AI)**
   - 既然已经集成了 `whisper.cpp`，未来可以考虑集成轻量级的本地 LLM（如 `Gemma 2B` 或 `Qwen 1.5B` 的 GGUF 格式），通过 `llama.cpp` 运行。实现**无网络状态下的基础语法纠错和词典查询**，这对于在地铁/飞机上学习的用户是巨大的加分项。
2. **Prompt 模板市场 (Prompt Engineering)**
   - 允许高级用户自定义 AI 分析的 System Prompt，甚至可以导出分享。比如某位老师调教出了极好的“德语从句分析 Prompt”，可以分享给其他用户。
3. **性能优化：长文本虚拟列表**
   - 在“阅读模式”或“词频 Top 200”列表中，如果项目文本极长，务必使用 `ListView.builder` 配合虚拟滚动，避免 Flutter 渲染掉帧。
4. **Anki 导出的增强 (Media Sync)**
   - 目前的 Anki TSV 导出是文本。如果能将**对应句子的音频切片（MP3）** 一起打包导出为 `.apkg` 格式，用户在 Anki 里就能直接听音复习，这会让你的应用在德语学习圈被“封神”。

### 💡 总结：产品定位的升华

目前的 DeutschFlow 是一个 **“超级听写与练习工具”**。
如果加上了上述的 **“语法可视化”、“复合词拆解”、“动态 Role-play”** 和 **“实景 OCR”**，它将进化为一个 **“全场景德语私教”**。

**建议的下一步迭代优先级 (Roadmap)：**
1. **P0 (核心体验)**：移动端德语快捷输入条、听写语义级容错、德汉词典接入。
2. **P1 (德语特色)**：复合词拆解、词性色彩编码、固定搭配提取。
3. **P2 (AI 扩展)**：文本降维改写、基于课文的动态对话、TTS 引擎集成。
4. **P3 (生态闭环)**：多端云同步、Anki 音频切片导出、B站/国内播客源支持。

这款应用的底子非常扎实，逻辑严密，技术选型现代。只要在“德语特有痛点”上再深挖一层，绝对有潜力成为多邻国 (Duolingo) 之外，中高级德语学习者必备的“硬核神器”！祝开发顺利！Viel Erfolg! 🚀
